# Paper 2

## A Distributed Data Engineering Framework for Real-Time Environmental Monitoring in Arctic Regions

This paper presents a distributed data architecture for Arctic environmental monitoring using a three-tier design with edge collection, message-based transmission, and centralized analytics. It emphasizes reliability under intermittent connectivity, heterogeneous sensor ingestion, time-series storage, and scalable query performance. The work is positioned as a practical systems framework for constrained remote deployments.

**Keywords:** distributed systems, data engineering, Arctic monitoring, MQTT, TimescaleDB, edge computing
