# UAF Research Papers

This repository contains two independent research papers developed in the context of graduate-level computer science research aligned with the University of Alaska Fairbanks.

## Papers

### 1. An AI-Powered Early Warning System for Predicting Infrastructure Failure in Remote Arctic Communities
This paper explores a hybrid machine learning framework for predictive maintenance in remote Arctic infrastructure settings. It focuses on early warning, anomaly detection, intermittent connectivity, and deployment realism in constrained environments.

**Themes:** predictive maintenance, anomaly detection, remote systems, edge-aware ML, infrastructure resilience

### 2. A Distributed Data Engineering Framework for Real-Time Environmental Monitoring in Arctic Regions
This paper proposes a distributed systems architecture for environmental monitoring in Arctic regions, with emphasis on intermittent connectivity, edge collection, time-series ingestion, and low-latency analytics.

**Themes:** distributed systems, data engineering, Arctic monitoring, MQTT, time-series databases, edge computing

## Why this repository exists
